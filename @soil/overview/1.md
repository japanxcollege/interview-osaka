# クイックスタートガイド

AIインタビューエディターへようこそ。このガイドでは、主な機能を簡潔に紹介します。

## 1. インタビューを開始する
- **ログイン/参加**: 名前を入力し、新規セッションの作成または既存セッションへの参加を行います。
- **マイク許可**: ブラウザのメッセージが表示されたら、マイクの使用を「許可」してください。

## 2. 録音と文字起こし
- **録音開始**: 画面上のマイクアイコン、または「Start Recording」をクリック。
- **リアルタイム表示**: 話した内容は即座に画面右側のパネルにテキスト化されます。

## 3. AIによるサポート
- **原稿自動生成**: 文字起こしがある程度（10件〜）たまると、AIが自動で記事のドラフトを生成します。
- **質問提案**: 会話の流れに応じて、AIが「次に聞くべきこと」を画面右側のパネルで提案します。

## 4. 編集と仕上げ
- **エディタ操作**: 中央のパネルで、文字起こしやAI生成された原稿を自由に加筆・修正できます。
- **自動保存**: 作業内容はリアルタイムで保存されます。
- **録音停止**: インタビューが終わったら「Stop Recording」をクリック。最後に全体要約が生成されます。

---
**Tips**: 動作が不安定な場合はページをリロードしてください。データは保持されます。
